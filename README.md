# caption-gallery

# How to run

- Clone repo
```bash
git clone https://github.com/opxz7148/caption-gallery
```

- Get into directory
```bash
cd caption-gallery
```

- Start virtual environment

**For macOS/Linux:**
```bash
source venv/bin/activate
```

**For Windows:**
```bash
venv\Scripts\activate
```

- Install dependencies
```bash
pip install -r requirements.txt
```

- Initialize DB
```bash
init.sh
```

- Run the application
```bash
python app.py
```

The app will be available at `http://localhost:8000`
